# 取り組みテーマ

高齢者の方で、病気なんじゃないかと意識する、副作用で飲みたくない等、薬に対して否定的。<br>
また、薬を飲むことを忘れてしまう方がいる。そのような高齢者に対して、楽しい注意喚起を行うことで薬を服用することに対する否定的な気持ちを減らしたい
<br><br><br>

# 概要

### 楽しい注意喚起を行うことによって、薬を服薬してもらう。

**流れ**

1. センサーで薬が飲まれたか検知する。

2. 薬が飲まれたら音楽を鳴らす、その後 1 へ戻る。薬が飲まれていなければ３へ

3. 王将を回す、音声を再生｛王手、王手、王手｝

4. 薬が飲まれたら停止、10 分程度で自動停止、自動停止の場合メール送信

**最初にメールアドレスの設定が必要**
